### Grid


#### Grid 和 Flexbox
```
flexbox 是一维布局，只能在一条直线上放置内容区块，有一个主轴和交叉轴

grid是一个二维布局，可以放置在任何地方
只是flexbox的一种升级

两个能很好的配合使用
```

- 1，一些概念
- 2，容器中的属性
- 3，CSS函数
- 4，网格项上的属性


#### 一些概念

- Grid Container 网格容器
```
display: grid;
所有网格项的父元素
```
- Grid Item 网格项
```
网格容器的子元素
```
- Grid Line 网格线
```
网格项的分界线
```
- Grid Track 网格轨道
```
两个相邻的网格线之间为网格轨道
```
- Grid Cell 网格单元
```
两个相邻的列网格线和两个相邻的行网格线组成的
```
- Grid Area 网格区域
```
四个网格线包围的总空间
```
